# Frontend Portfolio

Навчальні проекти, створені для відпрацювання навичок верстки та роботи з CSS-фреймворками.

## Основні проекти

### 1. Тематичний лендинг (House Targaryen)

- **Мета:** Практика використання препроцесора LESS та бібліотеки Materialize.
- **Технічна частина:**
  - Модульна структура LESS: змінні для кольорів та міксини для повторюваних стилів.
  - Використання JS-компонентів: Parallax, Modals, Toasts.
  - Адаптивність через вбудовану Grid-систему фреймворку.

### 2. Framework Showcase (Bootstrap & Materialize)

- **Мета:** Порівняння роботи з різними UI-бібліотеками.
- **Що зроблено:** Верстка однакових за структурою блоків (картки, навігація) за допомогою різних інструментів.

## Стек

HTML5, CSS3, LESS, JavaScript (Basic), Bootstrap 5, Materialize CSS, Git.
